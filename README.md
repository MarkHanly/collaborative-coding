# HDAT9400
A simple example illustrating some good-practice elements of collaborative coding

## Inputs:

* hospital.sas7bdat  
* survey.sas7bdat  


## Outputs:

* variable_summary.pdf  
* risk_by_age_sex.pdf

2.SummVars *Summarise the study variables*
